## Programmable timer with 4-channel PWM

Connections:

|	-	| Arduino NANO	|
|---------------|---------------|
| **LCD**	|		|
| D4		| D0/RX		|
| D5		| D1/TX		|
| D6		| D2		|
| D7		| D3		|
| E		| D4		|
| RW		| D5		|
| RS		| D6		|
|		|		|
| **Keyboard**	|		|
| RIGHT		| A0		|
| DOWN		| A1		|
| UP		| A2		|
| LEFT		| A3		|
|		|		|
|**PWM output**	|		|
| PWM0		| D13		|
| PWM1		| D12		|
| PWM2		| D11		|
| PWM3		| D10		|
